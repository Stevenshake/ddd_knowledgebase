# Блог на React

Современный блог с поддержкой Markdown, галереями изображений и админ-панелью.

## Функционал

### Для пользователей:
- **Главная страница** - просмотр всех статей в виде сетки или списка
- **Страница статьи** - полный просмотр статьи с баннером, контентом и галереей

### Для администратора:
- **Управление статьями** - добавление, удаление, перемещение статей (drag & drop)
- **Редактор статей** - создание и редактирование статей в формате Markdown
- **Галерея изображений** - загрузка картинок через выбор файла или drag & drop, сортировка

## Использование

### Просмотр статей
1. Откройте главную страницу `/`
2. Переключайтесь между режимами "Сетка" и "Список"
3. Кликните на статью для просмотра

### Создание статьи (Админ)
1. Перейдите в раздел "Админ" `/admin`
2. Нажмите "+ Добавить статью"
3. Заполните поля:
   - Название статьи
   - URL (slug) - генерируется автоматически
   - Баннер статьи (изображение)
   - Краткое описание
   - Содержимое в формате Markdown
   - Галерея изображений (опционально)
4. Нажмите "Опубликовать статью"

### Редактирование статьи
1. В админ-панели нажмите "Редактировать" у нужной статьи
2. Внесите изменения
3. Нажмите "Сохранить изменения"

### Управление порядком статей
- В админ-панели перетаскивайте статьи за иконку ☰ для изменения порядка отображения

## Формат Markdown

В статьях поддерживается стандартный синтаксис Markdown:

```markdown
# Заголовок 1 уровня
## Заголовок 2 уровня

**Жирный текст** и *курсив*

- Маркированный список
- Элемент списка

1. Нумерованный список
2. Второй элемент

[Ссылка](https://example.com)

![Изображение](url)

> Цитата

`Код в строке`

```javascript
// Блок кода
const hello = "world";
```
```

## Хранение данных

В текущей версии данные хранятся в памяти приложения (React Context). 
Для постоянного хранения необходимо подключить бэкенд или использовать localStorage.

### Структура статьи

```javascript
{
  id: string,              // Уникальный ID
  slug: string,            // URL статьи
  title: string,           // Название
  banner: string,          // Путь к баннеру
  excerpt: string,         // Краткое описание
  content: string,         // Markdown контент
  gallery: array,          // Массив изображений галереи
  order: number,           // Порядок отображения
  createdAt: string        // Дата создания
}
```

## Технологии

- **React** - UI библиотека
- **React Router** - Роутинг
- **@dnd-kit** - Drag & Drop для галереи и сортировки статей
- **react-markdown** - Рендеринг Markdown
- **Vite** - Сборщик проектов

## Развитие проекта

Для добавления постоянного хранения статей:

1. Создайте API для CRUD операций со статьями
2. Настройте хранение файлов (баннеры, галерея)
3. Обновите `BlogContext.jsx` для работы с API
4. Добавьте аутентификацию для админ-панели
